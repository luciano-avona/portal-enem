# Portal do Candidato ENEM

## 📌 Sobre o Projeto

O Portal do Candidato ENEM é uma aplicação web desenvolvida para a disciplina de Versionamento do curso Técnico em Desenvolvimento de Sistemas (M-TEC 3 DS).

O sistema simula uma área interna de candidatos do ENEM, permitindo a navegação entre páginas, consulta de cursos por área do conhecimento e realização de inscrição para a prova.

O projeto foi desenvolvido utilizando HTML, CSS, JavaScript e Git/GitHub, aplicando conceitos de versionamento, manipulação do DOM e validação de formulários.

---

## 🚀 Funcionalidades

### Painel Principal

* Mensagem de boas-vindas ao usuário.
* Navegação através de cards.
* Acesso rápido às principais funcionalidades do sistema.

### Consulta de Cursos

* Listagem de cursos organizados por área do conhecimento.
* Utilização de Array Bidimensional.
* Filtro dinâmico utilizando JavaScript.
* Atualização dos resultados sem recarregar a página.

### Inscrição na Prova

* Seleção de língua estrangeira.
* Escolha de estado e cidade.
* Atendimento especializado opcional.
* Exibição dinâmica de campos através da manipulação do DOM.
* Validação de campos obrigatórios.
* Exibição de mensagens de erro e sucesso.
